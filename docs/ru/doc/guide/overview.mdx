## Быстрый старт

### Локальная установка VoceSpace в один клик для вашего удобства

Пожалуйста, замените `your_ip` на ваш реальный `IP`

```bash
curl https://vocespace.com/local_deploy/local_deploy.sh|bash --host-ip your_ip
```
