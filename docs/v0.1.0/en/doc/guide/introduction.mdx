[Vocespace](https://github.com/Privoce/vocespace-client)是面向现代企业的安全通信平台, 我们将帮助您在子域下托管自己的安全视频和音频会议平台,并提供您自己的徽标和品牌。使用企业级加密完全控制您的数据。


## 功能

以下是Vocespace具备的基本功能

- 设备设置、设备选择、设备调整
- 麦克风、摄像头、屏幕分享
- 房间管理
- 用户状态管理、状态分组
- 虚拟形象
- wave提醒
- 聊天与文件传输
- 国际化
- e2ee加密，隐私保护
