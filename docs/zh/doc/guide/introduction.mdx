[VoceSpace](https://github.com/Privoce/vocespace-client)是面向现代企业的安全通信平台, 我们将帮助您在子域下托管自己的安全视频和音频会议平台,并提供您自己的徽标和品牌。使用企业级加密完全控制您的数据。

在当今数字化时代，企业通信的需求日益增长，不仅要求高效便捷，更要确保数据安全与隐私保护。VoceSpace 应运而生，作为一款基于 LiveKit 构建的现代企业安全通信平台，VoceSpace 为企业提供了一个安全、可定制且功能强大的视频和音频会议解决方案。

## VoceSpace 是什么

VoceSpace 是一款专为现代企业设计的安全通信平台。它基于 LiveKit 构建，利用 LiveKit 的强大性能和灵活性，为企业提供了一个稳定、高效的通信环境。VoceSpace 不仅支持视频和音频会议，还具备一系列高级功能，如设备设置、屏幕分享、多用户鼠标同屏、虚拟形象等，能够满足企业在多种场景下的通信需求。

VoceSpace 的核心优势在于其强大的安全性和隐私保护。通过企业级加密技术，VoceSpace 确保用户数据完全可控，从会议内容到文件传输，都经过端到端加密（E2EE），有效防止数据泄露和未经授权的访问。此外，VoceSpace 提供私有部署选项，允许企业在自己的子域下托管平台，并使用自己的徽标和品牌，进一步增强企业的数据控制权和品牌形象。

## VoceSpace 适合什么场景

VoceSpace 适用于多种企业｜个人场景，包括但不限于：

1. **远程办公**：随着远程工作模式的普及，企业需要一个稳定、安全的通信平台来支持员工之间的协作。VoceSpace 提供高清视频和音频会议功能，支持屏幕分享和多用户鼠标同屏操作，使远程团队能够高效沟通和协作，如同在办公室一样。
2. **在线教育**：教育机构可以利用 VoceSpace 进行在线教学，通过虚拟形象和互动功能，增强学生的学习体验。同时，VoceSpace 的隐私保护功能确保学生和教师的个人信息安全。
3. **医疗咨询**：在医疗领域，VoceSpace 的 E2EE 加密和隐私保护功能尤为重要。医生可以通过 VoceSpace 与患者进行远程会诊，确保患者的医疗信息不被泄露。
4. **企业培训**：企业可以使用 VoceSpace 进行内部培训和研讨会，支持大量用户同时在线，提供实时互动和文件传输功能，提高培训效果。
5. **国际会议**：VoceSpace 支持国际化，能够满足跨国企业或组织的通信需求，确保不同语言和地区的用户都能顺畅使用平台。
6. **游戏开黑**：组队游戏时使用 VoceSpace 感受队友在你身边的开黑时的同座感受。
